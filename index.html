PUZZLE GAME

Puzzle games focus on completion, which requires players to solve a logic puzzle or navigate a challenge to progress to the next, more difficult challenge. Depending on the puzzle game type, it may involve playing with multiple players, but typically, puzzle games are played solo.

Instruction:

  1. First Select the grid size
  2. Either start with default image or select a image from your device.
  3. You can see the solved image in left side.
  4. Move images blocks on board by clicking on blocks
  5. You can move only left, right up and down
  6. On right side, you can see your count
  7. When blocks are arranged in right way. You win
